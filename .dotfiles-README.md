# Setting up bare repo in home folder

## Bootstrap repo
1. git clone --bare https://github.com/Kevy42/dotfiles.git $HOME/.dotfiles
2. alias dotfiles='/usr/bin/git --git-dir=$HOME/.dotfiles --work-tree=$HOME'
3. dotfiles config --local status.showUntrackedFiles no
4. Pull or checkout files

## Dependencies
1. Git
2. Nerd Font (check kitty.conf)
3. Kitty
4. fzf
5. Commmand-line copying utility (e.g wl-copy) 
6. Tmux
7. tree-sitter-cli
8. wget
9. npm
10. ripgrep
11. Neovim
12. VsCode

## Post-setup
1. Create ~/.gitconfig-private and add the following:
    ```
    [user]
            email = [EMAIL]
            name = [FULL NAME]
    ```
2. Create and configure ~/.ssh/personal_config & ~/.ssh/work_config (see ~/.ssh/config)
3. Run "kitten theme" to generate current-theme.conf
4. Bootstrap Tmux TPM and install plugins
